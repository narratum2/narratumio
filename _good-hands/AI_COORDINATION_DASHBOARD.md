## 🎓 Learning & Knowledge Base

### Common Patterns Discovered

1. **Spacing Fix Template**
   ```tsx
   // Replace: mt-10
   // With: mt-12 (48px = 8px grid)
   ```

2. **Color Fix Template**
   ```tsx
   // Replace: bg-[#f5f5f5]
   // With: bg-porcelain
   ```

3. **Focus State Template**
   ```tsx
   // Add: focus:ring-2 focus:ring-gold focus:outline-none
   ```
